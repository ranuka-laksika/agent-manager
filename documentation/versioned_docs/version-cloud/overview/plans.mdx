# Plans & Pricing

WSO2 Agent Manager is available on a **free plan** during the current early access period, with paid plans coming soon.

## Free Plan

The free plan gives you full access to the platform with the following limits:

| Resource | Limit |
|----------|-------|
| Platform-Hosted Agents | 5 |
| Externally-Hosted Agents | 5 |

### What's included

- Full agent lifecycle management (build, deploy, scale)
- Observability — traces, metrics, and logs
- Continuous evaluation and monitors
- LLM provider integrations
- AI gateway support
- API key security for agent endpoints

## Paid Plans

Paid plans with higher limits and additional features are coming soon. [Contact us](mailto:agent-manager@wso2.com) if you need higher limits in the meantime.
